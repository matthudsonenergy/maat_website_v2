## How to Embed on Google Sites

1. Open your Google Site in Edit mode.
2. Navigate to the target page (e.g., Technology > Unified Platform).
3. For each snippet in order:
   - Click **Insert** → **Embed** → **Embed code**.
   - Paste the full HTML content of the snippet file.
   - Click **Next** → **Insert**.
   - Resize the embed block to fill the page width.
   - Adjust the height to match the suggested values above (may vary by screen).
4. Repeat for all 6 snippets, stacking them vertically.
5. **Publish** the site.

## Design Notes

- All styles are **inline** for Google Sites compatibility (no external CSS).
- Fonts fall back to `Segoe UI → Helvetica Neue → Arial → sans-serif` to match the clean, professional tone of the existing site.
- Color palette derived from the teal/green accent family visible on maatenergy.com (technology pages, application badges, footer).
- Cards use subtle `border-left` accents and light surface fills instead of heavy shadows, matching the site's understated style.
- The "Build Notes" sidebar from the original has been intentionally **excluded** — it was internal guidance, not visitor-facing content.
- All snippets are responsive and collapse gracefully on mobile.
- The CTA banner links to the existing `/contact` page.
